# Carpeta de fotos — Aura Experience

Usá esta carpeta para **intercambiar, ver, sustituir o eliminar** fotos del proyecto.

## Cómo usarla desde GitHub (web)
- **Subir fotos:** entrá a esta carpeta en GitHub → botón **Add file → Upload files** → arrastrá tus imágenes → **Commit changes**.
- **Ver / descargar:** hacé clic en cualquier imagen para verla; con **Download** la bajás.
- **Eliminar:** abrí la imagen → ícono de tacho (🗑) → **Commit changes**.

## Recomendaciones
- Formato **JPG** o **PNG**. Para que no pesen: hasta ~1600 px de ancho.
- Nombralas claro, por ejemplo: `martina-1.jpg`, `martina-2.jpg`.

> Nota: las fotos que ven las modelos y clientes en la web se cargan desde el panel
> (Publicar / Editar perfil). Esta carpeta es tu espacio de trabajo para organizarlas.
